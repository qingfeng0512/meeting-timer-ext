# 计时器 Chrome 扩展

一个简洁实用的倒计时工具扩展，支持多种时间预设、自定义时间和拖拽定位。

## ✨ 功能特点

### 🎯 多种时间预设
- **秒级预设**：10秒、20秒、30秒
- **分钟预设**：1分钟、2分钟、3分钟、5分钟、10分钟、30分钟
- **自定义时间**：支持格式如 `30s`、`1m`、`90`（默认秒）

### 🎨 薄荷绿主题
- 清新的薄荷绿配色方案
- 现代化渐变设计
- 流畅的动画效果

### 🖱️ 拖拽定位
- 悬浮窗支持拖拽到任意位置
- 记住拖拽位置
- 平滑拖拽体验

### 🎤 语音提示
- 倒计时结束时语音朗读"倒计时结束"
- 无需音效文件，纯 JavaScript 实现
- 支持中文语音合成

### 📊 详细信息显示
- 倒计时进度条
- 百分比进度显示
- 状态指示器（运行中/已暂停/已完成）

### 🔔 提醒功能
- 桌面通知
- 页面悬浮显示
- 倒计时状态实时同步

## 🚀 安装使用

### 1. 开发者模式安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的"**开发者模式**"
3. 点击"**加载已解压的扩展程序**"
4. 选择项目文件夹
5. 扩展安装完成！

### 2. 使用方法

#### 方法一：弹窗模式
1. 点击浏览器工具栏中的扩展图标
2. 在弹窗中选择时间或输入自定义时间
3. 点击"**开始**"按钮

#### 方法二：侧边栏模式
1. 右键点击扩展图标
2. 选择"**在侧边栏中显示**"
3. 在侧边栏中操作

### 3. 悬浮窗操作
- **拖拽移动**：按住悬浮窗并拖动到任意位置
- **自动显示**：选择时间后自动在页面左上角显示
- **实时更新**：倒计时过程中实时更新

## 📝 支持的时间格式

```
30s   → 30秒
1m    → 1分钟
90    → 90秒（默认单位）
5m    → 5分钟
600   → 600秒
```

| 按钮 | 功能 |
|------|------|
| 快速设置 | 选择预设时间（10秒-30分钟） |
| 开始 | 开始倒计时 |
| 暂停 | 暂停倒计时 |
| 重置 | 重置到初始时间 |
| 设置 | 应用自定义时间 |
